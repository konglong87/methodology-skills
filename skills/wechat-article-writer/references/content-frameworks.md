# Content Frameworks

## Knowledge Type (知识型)

**框架**: 痛点-解决方案-案例-行动建议

**适用**: 工具推荐、资源盘点、干货分享

**示例**:
1. 开头: 描述效率低下的痛点
2. 方案: 介绍5个AI工具
3. 案例: 每个工具的实际应用
4. 行动: 如何选择和使用

## Tutorial Type (教程型)

**框架**: 问题-方案-步骤-总结

**适用**: 操作指南、教程、how-to

**示例**:
1. 问题: 想要完成某个任务
2. 方案: 使用XX工具/方法
3. 步骤: 详细的操作步骤
4. 总结: 关键要点回顾

## Emotional Type (情感型)

**框架**: 场景-冲突-转折-启示

**适用**: 个人经历、感悟分享、情感故事

**示例**:
1. 场景: 设定故事背景
2. 冲突: 遇到的困难和挑战
3. 转折: 如何克服和改变
4. 启示: 得到的感悟和成长

## Review Type (测评型)

**框架**: 背景-对比维度-结论-建议

**适用**: 产品对比、方案评测、优劣分析

**示例**:
1. 背景: 为什么需要对比
2. 维度: 从多个角度对比
3. 结论: 各自的优劣
4. 建议: 如何选择
