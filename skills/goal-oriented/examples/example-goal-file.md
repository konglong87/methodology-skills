## 目标提取（SMART）
- **Specific（具体）**: 实现 goal-oriented skill 的自动目标追踪和验证机制

- **Achievable（可实现）**: 使用 Python 脚本辅助，技术可行
- **Relevant（相关）**: 防止 AI 执行过程中丢失目标上下文
- **Time-bound（时限）**: 本次会话完成
