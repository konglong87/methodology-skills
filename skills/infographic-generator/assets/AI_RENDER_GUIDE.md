# AI驱动渲染指南

本指南说明如何使用AI驱动渲染功能，从自然语言描述直接生成PNG图片。

## 概述

AI驱动渲染功能允许您使用自然语言描述需求，自动生成JSON配置并渲染PNG图片，无需手动创建配置文件。

## 使用方法

### 基本用法

```bash
node ai-render.js "请帮我生成一个关于Python编程语言的信息图"
```

### 指定输出路径

```bash
node ai-render.js "请帮我生成一个关于Python编程语言的信息图" output/python.png
```

## 示例

### 示例1：知识科普信息图

```bash
node ai-render.js "请帮我生成一个关于Python编程语言的信息图，使用知识科普模板，包含4个核心特点：语法简洁、应用广泛、生态丰富、跨平台，使用极简风格"
```

### 示例2：SWOT分析信息图

```bash
node ai-render.js "请帮我生成一个关于人工智能的SWOT分析信息图"
```

### 示例3：流程图信息图

```bash
node ai-render.js "请帮我生成一个关于软件开发流程的流程图信息图"
```

## 输出

AI驱动渲染会生成两个文件：

1. **PNG图片**：渲染好的信息图
2. **JSON配置**：自动生成的配置文件（temp-config.json）

## 功能特点

- ✅ **自然语言输入**：使用自然语言描述需求
- ✅ **自动生成配置**：AI自动生成JSON配置
- ✅ **自动渲染**：自动渲染PNG图片
- ✅ **保存配置**：自动保存JSON配置文件
- ✅ **灵活输出**：支持指定输出路径

## 工作流程

1. **接收自然语言输入**：解析自然语言描述
2. **生成JSON配置**：AI生成JSON配置
3. **保存配置文件**：保存JSON配置到文件
4. **加载模板**：加载指定的模板
5. **渲染信息图**：渲染信息图
6. **保存PNG**：保存PNG图片

## 支持的模板

- `knowledge`：知识科普模板（默认）
- `swot`：SWOT分析模板
- `process`：流程图模板

## 支持的风格

- `极简`：极简风格
- 默认风格

## 注意事项

1. **自然语言描述**：请提供清晰的自然语言描述，包括主题、模板、风格等信息
2. **输出路径**：输出路径是可选的，如果不指定，会使用默认路径
3. **配置文件**：生成的JSON配置文件会保存为`temp-config.json`，可以手动修改后重新渲染

## 高级用法

### 修改生成的配置

1. 运行AI驱动渲染生成配置文件
2. 手动修改`temp-config.json`
3. 使用`auto-render.js`重新渲染：

```bash
node auto-render.js temp-config.json
```

### 批量生成

创建一个脚本批量生成多个信息图：

```bash
#!/bin/bash

# 批量生成脚本
node ai-render.js "请帮我生成一个关于Python编程语言的信息图"
node ai-render.js "请帮我生成一个关于JavaScript编程语言的信息图"
node ai-render.js "请帮我生成一个关于Java编程语言的信息图"

echo "批量生成完成！"
```
