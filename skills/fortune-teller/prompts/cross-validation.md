# 交叉验证Prompt (5轮验证)

## 任务
对四个派系的分析结论进行交叉验证，确保结论可靠。

## 输入
- 紫微斗数结论：{ziweiConclusion}
- 八字结论：{baziConclusion}
- 盲派结论：{mengpaiConclusion}
- 南北派结论：{nanbeipaiConclusion}

## 验证流程

### 第1轮：各派系独立分析验证

**检查项：**
- 每条结论是否附带「依据来源」
- 结论是否符合该派系的基本原理
- 数据是否准确无误

**评分标准：**
- 有依据来源：合格
- 无依据来源：需补充
- 依据错误：需修正

### 第2轮：派系交叉验证

**验证项：**

1. **紫微斗数 vs 八字**
   - 命宫主星 vs 日主特性
   - 四化飞星 vs 十神
   - 大运走势一致性

2. **紫微斗数 vs 盲派**
   - 格局 vs 用神喜忌
   - 调候 vs 四化
   - 大运转折点

3. **八字 vs 南北派**
   - 用神一致性
   - 调候用神 vs 扶抑用神
   - 格局判断

4. **盲派 vs 南北派**
   - 调候用神一致性
   - 格局 vs 用神
   - 应期判断

**标注：**
- ✓ 一致：结论相互印证
- ✗ 矛盾：结论相互冲突
- ? 待定：无法判断，需补充证据

### 第3轮：LLM自我校验

**对矛盾点的处理：**
- 列出矛盾点
- 提供3个可能的解释
- 选择最合理的解释
- 标注依据

**示例：**
```
矛盾点：紫微斗数判断财运好，八字判断偏财弱
可能解释：
1. 紫微斗数看的是整体运势，八字看的是具体年份
2. 大运不同，阶段不同
3. 紫微斗数的财帛宫 vs 八字的财星定义不同
选择：解释1，依据：两派侧重点不同
```

### 第4轮：典籍依据核查

**核查典籍：**
- 紫微斗数：《紫微斗数全书》《斗数全书》
- 八字：《穷通宝鉴》《滴天髓》《三命通会》《子平真诠》
- 盲派：盲派秘传
- 南北派：南北派秘传

**依据等级：**
- ★★★ 确定：多派系一致，典籍有明确记载
- ★★ 较确定：单派系结论，有典籍支撑，无明显矛盾
- ★ 推测：无典籍直接依据，逻辑推演得出
- ⚠ 存疑：派系结论矛盾，无法确认

### 第5轮：结论一致性确认

**计算一致性评分：**

1. 统计所有结论
2. 统计一致结论数量
3. 计算一致性比例
4. 给出风险提示

**一致性等级：**
- 90-100%：结论高度一致，可信度高
- 70-89%：结论基本一致，有小矛盾
- 50-69%：结论存疑，需补充证据
- <50%：主要矛盾，建议标注为「仅供参考」
