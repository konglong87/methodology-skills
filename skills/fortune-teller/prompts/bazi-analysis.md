## 核心知识体系

### 一、真太阳时的重要性

**⚠️ 关键：算命必须使用真太阳时，而非北京时间！**

真太阳时 = 北京时间 + 经度修正
- 北京（东经120°）：无需修正
- 上海（东经121.5°）：+6分钟
- 广州（东经113°）：-28分钟
- 成都（东经104°）：-64分钟
- 乌鲁木齐（东经87.7°）：-129分钟

**举例：**
- 出生北京时间14:00，出生地广州（113°）
- 真太阳时 = 14:00 + (-28分钟) = 13:32
- 时辰判定：13:32属未时（13:00-15:00），而非申时

### 二、大运顺逆（男女不同）

**⚠️ 关键：大运排法，男女不同！**

- **男命阳干（甲丙戊庚壬）**：顺行（顺排）
- **男命阴干（乙丁己辛癸）**：逆行（逆排）
- **女命阳干**：逆行（逆排）
- **女命阴干**：顺行（顺排）

**举例：**
- 日主甲木（阳干），男命 → 大运顺排：乙丑、丙寅、丁卯...
- 日主乙木（阴干），男命 → 大运逆排：甲子、癸亥、壬戌...
- 日主甲木（阳干），女命 → 大运逆排
- 日主乙木（阴干），女命 → 大运顺排

### 三、五行生克关系

**天干五行：**
| 五行 | 天干 | 特性 |
|------|------|------|
| 木 | 甲（阳木）、乙（阴木） | 生发、成长 |
| 火 | 丙（阳火）、丁（阴火） | 热情、光明 |
| 土 | 戊（阳土）、己（阴土） | 稳重、包容 |
| 金 | 庚（阳金）、辛（阴金） | 刚毅、收敛 |
| 水 | 壬（阳水）、癸（阴水） | 智慧、流动 |

**地支五行：**
| 五行 | 地支 | 季节 |
|------|------|------|
| 木 | 寅、卯 | 春季 |
| 火 | 巳、午 | 夏季 |
| 土 | 辰、戌、丑、未 | 四季末 |
| 金 | 申、酉 | 秋季 |
| 水 | 亥、子 | 冬季 |

**生克关系：**
- 木生火，火生土，土生金，金生水，水生木
- 木克土，土克水，水克火，火克金，金克木

### 四、用神喜忌

**用神**：对日主最有利的五行
**喜神**：生助用神的五行
**忌神**：克制日主或用神的五行

**找用神的原则：**
1. **扶抑**：日主偏弱则扶，偏强则抑
2. **调候**：寒用火，热用水
3. **通关**：五行相克，取通关之神
4. **病药**：八字有病（缺陷），用神为药
