## 三维定位法（核心方法论）

盲派命理强调"三维定位"，从格局、流通、调候三个维度立体分析命局。

### 第一维度：格局定性

**格局定性判断流程**：

**步骤1：五行力量对比统计**
- 统计每个五行的数量（使用正确方法：只统计天干+藏干）
- 标注强弱等级（极旺、偏旺、中等、偏弱、极弱）

**步骤2：识别核心矛盾**
- 找出最旺的五行（主要矛盾）
- 找出最弱的五行（次要矛盾）
- 分析五行生克关系

**步骤3：格局命名规则**

| 最旺五行 | 对应十神 | 格局命名 |
|----------|----------|----------|
| 七杀 | 七杀 | 杀旺格局（杀旺身弱） |
| 财星 | 财星 | 财多格局（财多身弱） |
| 食伤 | 食伤 | 食伤旺格局 |
| 印星 | 印星 | 印旺格局 |
| 比劫 | 比劫 | 身旺格局 |

**重要提醒**：
- ⚠️ 格局命名以"最旺五行为准"
- ⚠️ 不能忽略最旺五行而关注次要五行
- ⚠️ 例如：七杀5个（极旺）> 财星3个（中等）→ 应为"杀旺格局"，而非"财多格局"

**案例示范**：

命局五行：
- 土（七杀）：5个 → 极旺
- 木（食神）：4个 → 偏旺
- 火（财星）：3个 → 中等
- 水（日主）：1个 → 极弱
- 金（印星）：1个 → 偏弱

**正确判断**：
- 最旺：七杀土（5个）
- 格局：**杀旺身弱**
- ❌ 错误判断：财多身弱（财星只有3个，不是最旺）

### 第二维度：流通路径

**五行生克的能量流动**：

**能量源头**：全局能量从何而起？

**流通路径描绘**：
- 金生水 → 水生木 → 木生火 → 火生土 → 土生金（良性循环）
- 或：木克土 → 土克水 → 水克火 → 火克金 → 金克木（制约关系）

**阻塞之处**：
- 刑冲合害导致的能量阻塞
- 五行断链导致的能量流失

**流通评语**：简述能量是否顺畅，是否形成良性循环

## 墓库能量开关理论

### 墓库基本概念

**四大墓库**：
- **辰（龙）**：水库，藏癸水、乙木、戊土
- **戌（狗）**：火库，藏丁火、辛金、戊土
- **丑（牛）**：金库，藏辛金、癸水、己土
- **未（羊）**：木库，藏乙木、丁火、己土

### 墓库的三种状态

**1. 开库状态（能量释放）**
- 被冲：辰戌冲、丑未冲 → 库门大开，能量爆发
- 被刑：辰辰自刑、丑戌刑、未戌刑 → 库门松动，能量溢出

**开库效应**：
- 库中十神能量释放
- 代表该方面的机遇或挑战
- 大运流年遇冲刑则开库

**2. 闭库状态（能量封存）**
- 被合：辰酉合、戌卯合、丑子合、未午合 → 库门紧闭，能量封存
- 三合局：申子辰合水、寅午戌合火、巳酉丑合金、亥卯未合木

**闭库效应**：
- 库中能量蓄势待发
- 代表该方面的潜力储备
- 需待时机开启

**3. 不开不闭（蓄势待发）**
- 无冲无合无刑无害
- 能量在库中酝酿
- 待大运流年触发

### 墓库的动态预测

**何时开库**：大运或流年遇冲刑时
**何时闭库**：大运或流年遇六合三合时

**案例示范**：
- 命局有辰，大运或流年遇戌 → 辰戌冲 → 水库开 → 释放癸水能量
- 命局有丑，大运或流年遇子 → 子丑合 → 金库闭 → 封存辛金能量

## 刑冲合会精析

### 六合关系

**六合组合**：
- 子丑合化土 → 合作、和解、牵绊
- 寅亥合化木 → 生扶、滋养、成长
- 卯戌合化火 → 激发、热情、创造
- 辰酉合化金 → 锻炼、提升、收获
- 巳申合化水 → 智慧、流动、变通
- 午未合化火/土 → 温暖、承载、包容

**合的含义**：
- 合多为贵，主和谐、合作
- 合去某字，则该字力量受制
- 合化成功需条件（月令、天干透出）

### 三合关系

**三合局**：
- 申子辰合水局 → 智慧、流动、灵活
- 寅午戌合火局 → 热情、行动、创造
- 巳酉丑合金局 → 锐利、决断、收获
- 亥卯未合木局 → 生长、仁慈、教育

**三合力量**：大于六合，能量更强

### 六冲关系

**六冲组合**：
- 子午冲：水火不容 → 情绪波动、内心矛盾
- 丑未冲：金木交战 → 健康隐忧、家庭纷争
- 寅申冲：木金相克 → 出行变动、驿马动荡
- 卯酉冲：木金相克 → 桃花纠纷、感情波折
- 辰戌冲：水库火库冲 → 财运起伏、墓库开启
- 巳亥冲：水火相克 → 事业变动、意外灾祸

**冲的含义**：
- 冲主动荡、变动、冲突
- 冲开墓库则能量释放
- 冲去某字则该字受损

### 相刑关系

**三刑**：
- 寅巳申三刑 → 恩将仇报、是非口舌
- 丑戌未三刑 → 恃势凌人、健康隐患
- 子卯相刑 → 无礼之刑、桃花纠纷

**自刑**：
- 辰辰自刑、午午自刑、酉酉自刑、亥亥自刑

**刑的含义**：
- 刑主矛盾、纠结、伤害
- 刑可开墓库
- 刑多主内心煎熬

### 相害关系

**六害组合**：
- 子未害、丑午害、寅巳害、卯辰害、申亥害、酉戌害

**害的含义**：
- 害主暗损、隐伤、小人
- 害不如冲刑明显，多为潜在影响

- 杀印相生权柄重，威震四方显英名
- 食神制杀化险夷，英雄豪杰世间稀
- 官印相生仕途顺，名利双收福禄增

### 墓库口诀
- 辰戌丑未墓库方，开闭冲合细推详
- 冲开墓库能量放，合闭墓库蓄势藏

### 流通口诀
- 五行流通生万物，阻滞不通百事殃
- 源头活水清如许，一脉相承福泽长

### 调候口诀
- 寒需火暖局方和，热要水调候始周
- 燥湿得宜福气厚，气候平衡命自优
