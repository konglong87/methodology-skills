## 使用说明

### 用户输入

用户用自然语言输入基本信息，例如：

```
帮我算命，张三 男 2025年3月15日 下午2点 北京
```

**必需信息（全部必填，不得省略）：**
- 姓名
- 性别（男/女）
- 出生日期（2025年3月15日 或 农历十月初六）
- 出生时辰（X点 或 上午X点、下午X点、晚上X点）
- 出生地点（北京、上海、广州等）

**⚠️ 重要：所有信息必须由用户提供，不允许使用默认值！**
如果用户没有提供任何一项，应反复询问，直到获取所有信息。

### AI工具执行流程

```
步骤1: AI工具识别意图
    用户: "帮我算命，张三 男 2025年3月15日 下午2点 北京"
    AI识别: 这是算命请求，触发 fortune-teller skill

步骤2: AI工具调用脚本
    Bash: node simple-fortune.js "张三 男 2025年3月15日 下午2点 北京"

步骤3: 脚本生成数据
    - 计算八字
    - 第1层验证
    - 生成增强数据
    - 输出JSON + Markdown + 提示词

步骤4: AI工具读取输出
    Read: output/张三_命盘数据.json
    Read: output/张三_分析框架.md

步骤5: AI工具自动分析（无需用户干预）
    - 读取提示词模板
    - 分析八字数据
    - 生成三层验证
    - 生成大运分析
    - 生成流年预测
    - 生成事业建议

步骤6: AI工具输出最终报告
    输出格式：Markdown
    字数：8000-12000字
```

- 命盘数据（四柱、五行、十神、大运等）

**3. 备份文件 → `backup/`**
- 用于存储旧版本、临时文件、调试数据等非必要文件
- 文件命名：`backup_[类型]_[时间戳].扩展名`
- 示例：`backup_old_20260316.json`

```javascript

['test', 'output', 'backup'].forEach(dir => {
  if (!fs.existsSync(dir)) {
    fs.mkdirSync(dir, { recursive: true });
  }
});
```

### 清理规则

**定期清理**：
- `test/` 目录：每周清理一次（删除7天前的文件）
- `backup/` 目录：每月清理一次（删除30天前的文件）
- `output/` 目录：永久保留，不自动清理

**手动清理**：
```bash
# 清理测试文件
rm test/test_*.json

# 清理备份文件
rm backup/backup_*.json
```

## 概述

FortuneTeller 是一个专业的命理分析工具，融合了四大主流命理派系：

- **紫微斗数**：十四主星、四化飞星、十二宫位
- **生辰八字**：干支五行、用神喜忌、大运流年
- **盲派**：调候+流通+格局三大核心
- **南北派**：独特用神法、命宫归宫、纳音取象

### 核心特性

1. **5轮验证机制**：确保结论可靠性
2. **时代背景贯穿**：大运流年分析包含时代背景
3. **交叉验证**：四大派系结论相互印证
4. **真太阳时计算**：根据出生地经度精确计算

## 工作原理

### 架构设计

FortuneTeller 采用 **计算引擎 + AI 智能分析** 的双层架构：

```
用户输入
  ↓
FortuneTeller (Node.js)
  ├─ 日期转换（农历/阳历）
  ├─ 真太阳时计算
  ├─ 八字排盘计算
  ├─ 紫微斗数排盘
  ├─ 盲派分析
  ├─ 南北派分析
  └─ 返回结构化数据（JSON）
      ↓
任意 AI 工具（LLM）
  ├─ Claude Code
  ├─ Cursor
  ├─ OpenCode
  ├─ OpenClaw
  ├─ Antigravity
  ├─ WorkBuddy
  ├─ Codex
  ├─ CodeBuddy
  ├─ QwenCode
  └─ 或任何其他 AI 编程助手
      ↓
  读取排盘数据
  ├─ 加载 prompts/*.md
  ├─ 执行5轮验证分析
  │   ├─ 第1轮：各派系独立分析
  │   ├─ 第2轮：派系交叉验证
  │   ├─ 第3轮：自我校验
  │   ├─ 第4轮：典籍依据核查
  │   └─ 第5轮：一致性确认
  └─ 生成完整命理报告
      ↓
用户获得报告
```

### AI 工具的角色

当 FortuneTeller 被任意 AI 工具调用时：
- **FortuneTeller**：负责所有复杂的命理计算（排盘、五行、四化等）
- **AI 工具**：负责所有需要智能分析的部分（性格分析、人生建议、综合推演等）
- **Prompts**：作为 AI 工具的知识库，提供命理分析的专业知识

**兼容性说明**：
- FortuneTeller 不依赖特定的 AI 工具
- 任何 LLM 都可以基于返回的 JSON 数据进行分析
- prompts/*.md 使用标准 Markdown 格式，所有 LLM 都能理解

## 使用方法

### 作为 AI 工具插件/功能使用（推荐）

当用户说"请帮我算命"时，AI 工具会自动：
1. 调用 FortuneTeller 进行排盘计算
2. 获取结构化的命理数据（JSON 格式）
3. 使用 prompts 进行5轮验证分析
4. 生成完整报告

**支持所有主流 AI 工具**：
- Claude Code、Cursor、OpenCode、OpenClaw
- Antigravity、WorkBuddy、Codex、CodeBuddy、QwenCode
- 或任何支持函数调用的 AI 编程助手

### 直接调用（开发者模式）

```javascript
const FortuneTeller = require('./fortune-teller');

const fortuneTeller = new FortuneTeller();

// 同步版本（只返回排盘结果，不含LLM分析）
const result = fortuneTeller.calculate({
  name: '张三',
  gender: '男',
  birthDate: { year: 1990, month: 8, day: 15 },
  birthTime: 14,
  location: '北京',
  calendarType: 'solar'
});

// result 包含：
// - 八字四柱、五行分布、日主强弱
// - 紫微斗数命宫、四化飞星
// - 盲派调候、流通、格局
// - 南北派用神分析
// - 时代背景数据

// 异步版本（返回排盘结果 + 空的分析框架）
const framework = await fortuneTeller.analyze({...});
// 注意：analyze() 只返回结构，需要 Claude 来填充实际分析内容
```
