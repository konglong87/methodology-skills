# FortuneTeller 算命Skill

> 融合紫微斗数、生辰八字、盲派、南北派四大派系的综合命理推算系统

## 概述

FortuneTeller 是一个专业的命理分析工具，融合了四大主流命理派系：

- **紫微斗数**：十四主星、四化飞星、十二宫位
- **生辰八字**：干支五行、用神喜忌、大运流年
- **盲派**：调候+流通+格局三大核心
- **南北派**：独特用神法、命宫归宫、纳音取象

### 核心特性

1. **5轮验证机制**：确保结论可靠性
2. **时代背景贯穿**：大运流年分析包含时代背景
3. **交叉验证**：四大派系结论相互印证
4. **真太阳时计算**：根据出生地经度精确计算

## 触发条件

当用户提及以下关键词时自动触发：
- 算命
- 八字算命
- 紫微斗数
- 命理分析
- 算一卦
- 看命
- 批八字
- 排盘

## 用户输入要求

请用户提供以下信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 姓名 | 是 | 用于报告称呼 |
| 性别 | 是 | 男/女，影响大运顺逆 |
| 出生日期 | 是 | 支持农历或阳历 |
| 出生时辰 | 是 | 24小时制（0-23） |
| 出生地 | 否 | 用于真太阳时计算 |

### 示例输入

```
姓名：张三
性别：男
出生日期：1990年8月15日（阳历）
出生时辰：14时
出生地：北京
```

或

```
姓名：李四
性别：女
出生日期：1995年农历正月初三
出生时辰：上午10点
出生地：上海
```

## 输出内容

系统会生成完整的命理分析报告，包含以下章节：

1. **基础信息** - 姓名、性别、出生日期、真太阳时
2. **紫微斗数分析** - 命宫主星、四化飞星、十二宫位、大运走势
3. **八字命理分析** - 四柱干支、五行分布、日主强弱、大运流年
4. **盲派分析** - 调候、流通、格局
5. **南北派分析** - 独特用神法、命宫归宫、纳音取象
6. **综合交叉验证** - 四大派系结论交叉验证
7. **时代背景分析** - 出生时期、大运流年的时代背景
8. **人生建议** - 职业方向、性格分析（需要LLM分析）
9. **免责声明** - 仅供参考，娱乐而已

### 数据

- `data/era-background.json` - 时代背景数据（1960-2035）

## 工作原理

### 架构设计

FortuneTeller 采用 **计算引擎 + AI 智能分析** 的双层架构：

```
用户输入
  ↓
FortuneTeller (Node.js)
  ├─ 日期转换（农历/阳历）
  ├─ 真太阳时计算
  ├─ 八字排盘计算
  ├─ 紫微斗数排盘
  ├─ 盲派分析
  ├─ 南北派分析
  └─ 返回结构化数据（JSON）
      ↓
任意 AI 工具（LLM）
  ├─ Claude Code
  ├─ Cursor
  ├─ OpenCode
  ├─ OpenClaw
  ├─ Antigravity
  ├─ WorkBuddy
  ├─ Codex
  ├─ CodeBuddy
  ├─ QwenCode
  └─ 或任何其他 AI 编程助手
      ↓
  读取排盘数据
  ├─ 加载 prompts/*.md
  ├─ 执行5轮验证分析
  │   ├─ 第1轮：各派系独立分析
  │   ├─ 第2轮：派系交叉验证
  │   ├─ 第3轮：自我校验
  │   ├─ 第4轮：典籍依据核查
  │   └─ 第5轮：一致性确认
  └─ 生成完整命理报告
      ↓
用户获得报告
```

### AI 工具的角色

当 FortuneTeller 被任意 AI 工具调用时：
- **FortuneTeller**：负责所有复杂的命理计算（排盘、五行、四化等）
- **AI 工具**：负责所有需要智能分析的部分（性格分析、人生建议、综合推演等）
- **Prompts**：作为 AI 工具的知识库，提供命理分析的专业知识

**兼容性说明**：
- FortuneTeller 不依赖特定的 AI 工具
- 任何 LLM 都可以基于返回的 JSON 数据进行分析
- prompts/*.md 使用标准 Markdown 格式，所有 LLM 都能理解

## 使用方法

### 作为 AI 工具插件/功能使用（推荐）

当用户说"请帮我算命"时，AI 工具会自动：
1. 调用 FortuneTeller 进行排盘计算
2. 获取结构化的命理数据（JSON 格式）
3. 使用 prompts 进行5轮验证分析
4. 生成完整报告

**支持所有主流 AI 工具**：
- Claude Code、Cursor、OpenCode、OpenClaw
- Antigravity、WorkBuddy、Codex、CodeBuddy、QwenCode
- 或任何支持函数调用的 AI 编程助手

### 直接调用（开发者模式）

```javascript
const FortuneTeller = require('./fortune-teller');

const fortuneTeller = new FortuneTeller();

// 同步版本（只返回排盘结果，不含LLM分析）
const result = fortuneTeller.calculate({
  name: '张三',
  gender: '男',
  birthDate: { year: 1990, month: 8, day: 15 },
  birthTime: 14,
  location: '北京',
  calendarType: 'solar'
});

// result 包含：
// - 八字四柱、五行分布、日主强弱
// - 紫微斗数命宫、四化飞星
// - 盲派调候、流通、格局
// - 南北派用神分析
// - 时代背景数据

// 异步版本（返回排盘结果 + 空的分析框架）
const framework = await fortuneTeller.analyze({...});
// 注意：analyze() 只返回结构，需要 Claude 来填充实际分析内容
```

## 准确率说明

本系统基于传统命理典籍开发，包括：
- 《穷通宝鉴》《滴天髓》《三命通会》《子平真诠》《千里命稿》《五行精纪》（八字）
- 《紫微斗数全书》《紫微斗数精成》《斗数全书》《飞星紫微斗数全书》（紫微斗数）
- 盲派秘诀

**重要提示**：
- 命理学本质是统计学，没有100%准确的算命
- 能有70%准确率的已经是顶尖高手
- 本报告仅供参考，娱乐而已
- 三分天注定，七分靠打拼
- 一名二运三风水，不要相信宿命论
- 只要心善、努力和乐观，就是好命

## 注意事项

1. **日期格式**：支持农历和阳历，系统会自动转换并验证
2. **真太阳时**：提供出生地可以获得更准确的时辰
3. **验证机制**：系统会进行2遍日期验证确保准确
4. **大运流年**：分析会考虑对应的时代背景
